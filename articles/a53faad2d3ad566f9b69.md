結構簡単に出来るみたいです。
当初は素直にlist作ってその中にtargetが存在するかチェックするループを使って…としていたんですが、数によってはめちゃくちゃ重くなるforを実行してしまいます。
これがイケてないなぁ、と思った時はこういう書き方にすると速くなるようです。

``` 重複除外.py 
def uniq(target):
    if cache.has_key(target) == False:
        cache[target] = target
        return False
    else:
        return True

cache = {}
for target in (hoge, fuga...):
   if uniq(target) == True:
       print '登録済み'
   else:
       print '登録しました'
```
